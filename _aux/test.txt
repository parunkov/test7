Во вкладке "Main" доступен макет одной страницы вымышленного сайта. Выполните вёрстку и отправите на почту job@mstagency.ru с темой “Фамилия Имя - Выполненное тестовое задание на Frontend разработчика” в письмо прикрепите ссылку на GitHub с проектом. В репозитории добавьте текстовый файл README.md с описанием работы, можно дополнительно написать там с какими сложностями столкнулись при выполнении задания. Так же результат - build выложите на GitHub Pages, или Netlify, или Surge... добавив ссылку в описание.
Шрифты используются бесплатные и доступны по следующим ссылкам: "https://www.tinkov.info/gilroy.html", "https://fonts.google.com/specimen/Roboto".

В блоке с описанием текст, больше чем в три строчки, должен автоматически обрезаться с добавлением ссылки в виде блока с многоточием.

Пример как должно работать переключение между страницами и между экранами разделов на странице, показано через анимацию в презентационным режиме Figma. А так вот дополнительная ссылка на видео демонстрацию
https://drive.google.com/file/d/1WbJQ0TL8pss1yrmEGhSf1bsY58tTWyxc/view?usp=sharing

Приветствуется использование системы сборки проекта и/или фреймворки: React, Vue.js, Next.js, Nuxt.js. Так же плюсом будет использование TypeScript.
